# ربات کپشن‌نویس سالن زیبایی

ربات تلگرام برای تولید کپشن‌های جذاب برای سالن‌های زیبایی

## خدمات:
- تولید کپشن برای ناخن
- تولید کپشن برای مو
- تولید کپشن برای پوست
- تولید کپشن برای میکاپ

## استقرار روی Railway:
1. ریپازیتوری را fork کنید
2. در Railway پروژه جدید بسازید
3. متغیر TELEGRAM_BOT_TOKEN را تنظیم کنید
4. ربات آماده است!
